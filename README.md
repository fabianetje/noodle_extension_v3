# noodle_extension_v3
new working noodle extension for beat saber in january 2026

How to run:

1. Make sure antivirus is off otherwise the .dll wont get extracted.

2. download both files

3. put both files on the desktop

4. run noodle_extension_v3

5. noodle_extension_v3 will find you're beatsaber file and inject itself in it's root.
